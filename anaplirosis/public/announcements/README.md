# Φωνητικές Ανακοινώσεις Σχολικού Ρολογιού

Αυτός ο φάκελος περιέχει τα mp3 αρχεία για τις φωνητικές ανακοινώσεις του σχολικού ρολογιού.

## Απαιτούμενα Αρχεία

Χρειάζεσαι τα παρακάτω mp3 αρχεία:

### Γενικά
- `morning.mp3` - Καλημέρα μήνυμα
- `end.mp3` - Τέλος σχολικού ωραρίου

### Περίοδοι (1-8)
- `period1.mp3` - "Ξεκινά η πρώτη περίοδος."
- `period2.mp3` - "Ξεκινά η δεύτερη περίοδος."
- `period3.mp3` - "Ξεκινά η τρίτη περίοδος."
- `period4.mp3` - "Ξεκινά η τέταρτη περίοδος."
- `period5.mp3` - "Ξεκινά η πέμπτη περίοδος."
- `period6.mp3` - "Ξεκινά η έκτη περίοδος."
- `period7.mp3` - "Ξεκινά η έβδομη περίοδος."
- `period8.mp3` - "Ξεκινά η όγδοη περίοδος."

### Διαλείμματα (1-3)
- `break1.mp3` - "Ξεκινά το πρώτοο διάλειμμα! Λίγη ξεκούραση…"
- `break2.mp3` - "Ξεκινά το δεύτεροο διάλειμμα! Πάμε λίγο να χαλαρώσουμε."
- `break3.mp3` - "Ξεκινά το τρίτοο διάλειμμα! Μην αργήσετε να επιστρέψετε…"

## Πώς να δημιουργήσεις τα αρχεία

### Επιλογή 1: Χρήση του generate-announcements.js script
```bash
node generate-announcements.js
```

Αυτό το script θα δημιουργήσει αυτόματα όλα τα mp3 αρχεία χρησιμοποιώντας Google Text-to-Speech.

### Επιλογή 2: Χειροκίνητη δημιουργία
1. Πήγαινε στο https://ttsmaker.com/ (ή άλλο online TTS service)
2. Επέλεξε φωνή: Ελληνικά
3. Αντίγραψε το κείμενο από το `../announcements.json`
4. Κατέβασε το mp3 και βάλτο στον φάκελο με το σωστό όνομα

### Επιλογή 3: Χρήση δικών σου ηχογραφήσεων
Μπορείς να ηχογραφήσεις τις ανακοινώσεις με τη φωνή σου και να τις αποθηκεύσεις ως mp3 με τα παραπάνω ονόματα.

## Σημαντικό
- Όλα τα αρχεία πρέπει να είναι σε μορφή MP3
- Προτείνεται bitrate: 128kbps
- Μέγεθος αρχείου: όσο το δυνατόν μικρότερο για γρήγορη φόρτωση
