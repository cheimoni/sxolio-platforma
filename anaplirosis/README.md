# Πρόγραμμα Αναπλήρωσης Ελληνικών Ονομάτων

Ένα React πρόγραμμα αυτόματης αναπλήρωσης ελληνικών ονομάτων με σύγχρονο και εύχρηστο interface.

## Χαρακτηριστικά

- 🔍 **Αυτόματη αναπλήρωση**: Προτάσεις ονομάτων καθώς πληκτρολογείτε
- ⌨️ **Πλοήγηση με πληκτρολόγιο**: Χρησιμοποιήστε τα βέλη ↑↓ και Enter
- 📱 **Responsive Design**: Λειτουργεί σε όλες τις συσκευές
- 🎨 **Σύγχρονο UI**: Όμορφο και εύκολο στη χρήση interface
- 📚 **Πλούσια βάση ονομάτων**: Περισσότερα από 400 ελληνικά ονόματα

## Εγκατάσταση

1. **Εγκαταστήστε τις εξαρτήσεις:**
   ```bash
   npm install
   ```

2. **Εκκινήστε την εφαρμογή:**
   ```bash
   npm start
   ```

3. **Ανοίξτε τον browser:**
   Πηγαίνετε στη διεύθυνση `http://localhost:3000`

## Χρήση

1. **Πληκτρολογήστε** τα πρώτα γράμματα ενός ονόματος
2. **Δείτε τις προτάσεις** που εμφανίζονται στο dropdown
3. **Επιλέξτε** ένα όνομα με:
   - Κλικ στο όνομα
   - Χρήση των βελών ↑↓ για πλοήγηση και Enter για επιλογή
4. **Ακυρώστε** με το πλήκτρο Escape

## Δομή Αρχείων

```
src/
├── App.js              # Κύριο component
├── App.css             # Στυλ για το App component
├── index.js            # Entry point
├── index.css           # Global styles
└── data/
    └── greekNames.js   # Βάση ελληνικών ονομάτων
```

## Τεχνολογίες

- **React 18** - JavaScript library για UI
- **CSS3** - Στυλιστικά με gradients και animations
- **Modern JavaScript** - ES6+ features

## Προσθήκη Νέων Ονομάτων

Για να προσθέσετε νέα ονόματα, επεξεργαστείτε το αρχείο `src/data/greekNames.js` και προσθέστε τα ονόματα στον πίνακα `greekNames`.

## Build για Production

```bash
npm run build
```

Αυτό θα δημιουργήσει ένα optimized build στο φάκελο `build/`.

## Scripts Διαθέσιμα

- `npm start` - Εκκινεί την εφαρμογή σε development mode
