# Λίστα Αντιστοιχίας - Κανονικά Τμήματα vs Στηρίξεις

## 📋 Τι υπάρχει στο HTML file (`tmimata kanonika .html`)

### ✅ Κανονικά Τμήματα (Regular Classes)
- **Μορφή**: `Τμήμα/Συνδιδασκαλία:   Α11`
- **Παραδείγματα**: 
  - Α11, Α21, Α22, Α23, Α24, Α31, Α32, Α41, Α42, Α43
  - Β1, Β31, Β32, Β33, Β41, Β51, Β52
  - Γ1, Γ31, Γ32, Γ41, Γ51, Γ52, Γ53
- **Χαρακτηριστικά**: 
  - Έχουν δικό τους κατάλογο μαθητών
  - Όνομα τμήματος: μόνο γράμμα + αριθμός (π.χ. "Γ53")

### 🆘 Στηρίξεις (Support Groups)
- **Μορφή 1**: `Τμήμα/Συνδιδασκαλία:   Στ. 11 (Γ41) Ψ.Α.`
  - Παραδείγματα: Στ. 11, Στ. 12, Στ. 13, Στ. 14, Στ. 15, Στ. 16, Στ. 17
  - Έχουν όνομα καθηγητή στο τέλος (π.χ. "Ψ.Α.", "Ρ.Γ.", "Μ.Α.")

- **Μορφή 2**: `Τμήμα/Συνδιδασκαλία:   Στ.Ο.1 (Γ41)`
  - Παραδείγματα: Στ.Ο.1, Στ.Ο.2, Στ.Ο.3, Στ.Ο.4, Στ.Ο.5, Στ.Ο.6, Στ.Ο.7, Στ.Ο.8, Στ.Ο.9, Στ.Ο.10
  - ΔΕΝ έχουν όνομα καθηγητή

- **Χαρακτηριστικά**:
  - Έχουν ΔΙΑΦΟΡΕΤΙΚΟΥΣ μαθητές από το κανονικό τμήμα
  - Όνομα στηρίξης: "Στ.Ο.X" ή "Στ. X" + τμήμα σε παρενθέσεις (π.χ. "Στ.Ο.4 (Γ1)")
  - Το τμήμα στις παρενθέσεις (π.χ. "(Γ1)") δείχνει από ποιο κανονικό τμήμα προέρχονται, αλλά έχουν ΔΙΑΦΟΡΕΤΙΚΟΥΣ μαθητές

## 🔍 Πρόβλημα που αντιμετωπίζουμε

### ❌ Τι συμβαίνει ΤΩΡΑ:
1. Ο χρήστης ζητάει: **"Στ.Ο.4 (Γ1)"** (στήριξη)
2. Η λογική εξάγει: **"Γ1"** (από τις παρενθέσεις)
3. Ψάχνει στο HTML: **"Τμήμα/Συνδιδασκαλία:   Γ1"** (κανονικό τμήμα)
4. Εμφανίζει: **Κατάλογο μαθητών του Γ1** (ΛΑΘΟΣ!)

### ✅ Τι ΠΡΕΠΕΙ να συμβαίνει:
1. Ο χρήστης ζητάει: **"Στ.Ο.4 (Γ1)"** (στήριξη)
2. Η λογική αναγνωρίζει: **"Στ.Ο.4 (Γ1)"** (ολόκληρο όνομα)
3. Ψάχνει στο HTML: **"Τμήμα/Συνδιδασκαλία:   Στ.Ο.4 (Γ1)"** (στήριξη)
4. Εμφανίζει: **Κατάλογο μαθητών της Στ.Ο.4** (ΣΩΣΤΟ!)

## 🛠️ Λύση

### Στο `StudentAttendanceList.js` - `parseTxtData()`:

**ΠΡΟΣΟΧΗ**: Όταν το `selectedClassName` είναι στήριξη (ξεκινάει με "Στ."), πρέπει:
1. ✅ Να κρατάμε ΟΛΟ το string (π.χ. "Στ.Ο.4 (Γ1)")
2. ✅ Να ψάχνουμε για ΟΛΟ το string στο HTML
3. ❌ ΝΑ ΜΗΝ εξάγουμε το τμήμα από τις παρενθέσεις

### Regex για αναγνώριση στηρίξεων:
```javascript
// Μορφή 1: "Στ. 11 (Γ41) Ψ.Α." ή "Στ. 11 (Γ41)"
const supportMatch1 = /^Στ\.\s*\d+\s*\([ΑΒΓ][0-9]+\)(?:\s+[Α-Ω]\.[Α-Ω]\.)?/i;

// Μορφή 2: "Στ.Ο.1 (Γ41)" ή "Στ.Ο.4 (Γ1)"
const supportMatch2 = /^Στ\.Ο\.\d+\s*\([ΑΒΓ][0-9]+\)/i;
```

### Λογική:
```javascript
// 1. ΠΡΩΤΑ: Ελέγχουμε αν είναι στήριξη
if (selectedClassName.match(/^Στ\./i)) {
  // Είναι στήριξη - χρησιμοποιούμε ΟΛΟ το όνομα
  classNameToSearch = selectedClassName.trim();
  isSupportClass = true;
} else {
  // Είναι κανονικό τμήμα - μπορούμε να εξάγουμε από παρενθέσεις αν χρειάζεται
  // ...
}
```

## 📊 Αντιστοιχίες

| Ζητούμενο από χρήστη | Τι πρέπει να βρούμε στο HTML | Κατηγορία |
|---------------------|------------------------------|-----------|
| `Γ1` | `Τμήμα/Συνδιδασκαλία:   Γ1` | Κανονικό τμήμα |
| `Γ53` | `Τμήμα/Συνδιδασκαλία:   Γ53` | Κανονικό τμήμα |
| `Στ.Ο.4 (Γ1)` | `Τμήμα/Συνδιδασκαλία:   Στ.Ο.4 (Γ1)` | Στήριξη |
| `Στ.Ο.3 (Γ53)` | `Τμήμα/Συνδιδασκαλία:   Στ.Ο.3 (Γ53)` | Στήριξη |
| `Στ. 11 (Γ41) Ψ.Α.` | `Τμήμα/Συνδιδασκαλία:   Στ. 11 (Γ41) Ψ.Α.` | Στήριξη |

## ⚠️ ΚΡΙΣΙΜΟ:
- **Στ.Ο.4 (Γ1)** έχει ΔΙΑΦΟΡΕΤΙΚΟΥΣ μαθητές από το **Γ1**
- **Στ.Ο.3 (Γ53)** έχει ΔΙΑΦΟΡΕΤΙΚΟΥΣ μαθητές από το **Γ53**
- Πρέπει να ψάχνουμε για το ΟΛΟΚΛΗΡΟ όνομα της στήριξης, ΟΧΙ μόνο το τμήμα!
